# Challenge 02 — Registration Form

A simple and structured **HTML Registration Form** built to practice core form elements and user input handling in HTML.

This challenge focuses on understanding how forms collect user information through different input fields and how semantic structure improves readability and accessibility.

---

## Objective

The goal of this challenge is to learn and practice:

- Creating HTML forms
- Taking user input using different input types
- Using labels properly
- Working with radio buttons and checkboxes
- Creating dropdowns and text areas
- Structuring forms professionally

---

## Concepts Practiced

- HTML Forms (`<form>`)
- Input Types (`text`, `email`, `password`, `number`)
- Labels (`<label>`)
- Radio Buttons
- Checkboxes
- Dropdown (`<select>`)
- Textarea (`<textarea>`)
- Submit Button
- Semantic HTML Structure

---

## Features

This registration form includes:

✅ Full Name Input  
✅ Email Address Input  
✅ Password Field  
✅ Gender Selection (Radio Buttons)  
✅ Skills Selection (Checkboxes)  
✅ Course/Interest Dropdown  
✅ Feedback Section  
✅ Submit Button
