# HTML Challenges

This folder contains coding challenges completed while learning HTML fundamentals.

The purpose of these challenges is to strengthen understanding of HTML concepts through hands-on implementation and problem-solving.

## Topics Covered

✔ Document Structure  
✔ Lists & Nested Lists  
✔ Tables  
✔ Forms  
✔ Semantic HTML  
✔ Accessibility  
✔ Navigation  
✔ Layout Structure  

## Challenge Approach

For each challenge:

- Understand the problem statement
- Build the structure using clean HTML
- Apply semantic elements whenever possible
- Maintain proper indentation
- Follow accessibility best practices
