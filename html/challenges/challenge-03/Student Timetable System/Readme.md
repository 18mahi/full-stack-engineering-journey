# Student Timetable System

A simple HTML-based **Student Timetable System** built to practice core table concepts in HTML.

## Objective

The goal of this challenge is to strengthen understanding of HTML table structures by creating a weekly study timetable.

## Concepts Practiced

- HTML Tables
- `<table>`
- `<tr>` (Table Row)
- `<th>` (Table Header)
- `<td>` (Table Data)
- `<caption>`
- `rowspan`
- `colspan`
- Table Structuring

## Features

- Weekly study timetable layout
- Subject-wise planning
- Time scheduling
- Priority tracking
- Clean table organization

## Challenge Outcome

Through this challenge, I practiced how to:

- Structure tabular data in HTML
- Organize schedules visually
- Use `rowspan` and `colspan`
- Improve semantic HTML understanding
