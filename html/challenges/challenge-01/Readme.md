# Challenge 01 — Personal Profile Card

## Objective
Build a simple personal profile card using only HTML.

## Concepts Practiced
- Basic HTML structure
- Semantic tags
- Headings and paragraphs
- Images
- Links
- Lists
- Buttons

## Features
- Profile image
- Name and bio section
- Skills list
- Social media links
- Contact button

## Technologies Used
- HTML5

## Learning Outcome
Learned how to structure a webpage using semantic HTML and organize personal information in a clean card layout.

## Future Improvements
- Add CSS styling
- Make responsive design
- Add hover effects
- Add dark mode
- Add animations using JavaScript
