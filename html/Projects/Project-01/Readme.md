# Student Learning Dashboard

A simple HTML-based Student Learning Dashboard created as part of my full stack learning journey.

## Project Overview

This project is a beginner-friendly dashboard that organizes student learning information in a structured web page format. It focuses on using semantic HTML elements to create a clean and meaningful layout.

## Features

- Header section with project title
- Navigation menu
- Main learning dashboard area
- Subject or skill sections
- Daily/weekly learning goals
- Progress or task overview
- Aside section for quick notes or reminders
- Footer section

## HTML Concepts Practiced

- `header`
- `nav`
- `main`
- `section`
- `article`
- `aside`
- `footer`
- Headings and paragraphs
- Lists
- Links
- Semantic page structure

## Project Goal

The goal of this project is to practice semantic HTML and build a structured page that can later be improved using CSS and JavaScript.
