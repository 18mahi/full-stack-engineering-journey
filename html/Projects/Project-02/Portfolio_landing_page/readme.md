# Portfolio Landing Page

A simple **Portfolio Landing Page** built using **HTML only**. This project showcases my personal profile, skills, goals, learning journey, and projects in a structured webpage format.

---

## Project Overview

This project is designed to create a personal portfolio webpage using only HTML. The main goal of this project is to practice HTML fundamentals, semantic tags, page structure, and content organization.

---

## Tech Stack

- HTML5

---

## Features

- Personal Introduction Section  
- About Me Section  
- Skills Section  
- Learning Goals Section  
- Education Details  
- Projects Showcase  
- Contact Information Section  
- Semantic HTML Structure  

---

## Concepts Practiced

- HTML Document Structure  
- Semantic Tags (`header`, `nav`, `main`, `section`, `article`, `footer`)  
- Headings and Paragraphs  
- Lists (`ul`, `ol`)  
- Tables  
- Hyperlinks (`a` tag)  
- Images  
- Forms Basics  
- Content Organization
